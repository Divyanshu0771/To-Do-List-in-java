# To-Do List in Java

A simple console-based Todo application built in Java.

## Features
- Add tasks
- View tasks
- Edit tasks
- Remove tasks
- Mark tasks as completed

## Concepts Used
- ArrayList
- Scanner (user input)
- Methods/Functions
- Loops and Conditionals

## How to Run
1. Compile: `javac TodoApp.java`
2. Run: `java TodoApp`
